# ㄱㅏㄴㅏ → 가나 | 한글 오토마타

> **자음·모음 시퀀스를 입력하면 완성형 한글로 조합해주는 한글 오토마타 구현**

---

## 📌 프로젝트 개요

키보드에서 자음과 모음을 순서대로 입력할 때 한글이 자동으로 조합되는 **한글 입력기의 핵심 오토마타 로직**을 Python으로 직접 구현했습니다.

- 초성·중성·종성 조합으로 완성형 유니코드 한글 생성
- **이중 모음** (ㅗ+ㅏ=ㅘ 등 9가지) 자동 합성
- **겹받침** (ㄹ+ㄱ=ㄺ 등 11가지) 자동 합성
- **백스페이스** (`<`) 처리 — 종성 → 중성 → 초성 순으로 되돌리기

---

## 🔤 한글 유니코드 조합 원리

```
유니코드 = 0xAC00 + (초성 인덱스 × 21 × 28)
                  + (중성 인덱스 × 28)
                  + 종성 인덱스
```

| 분류 | 개수 | 예시 |
|------|------|------|
| 초성 (CHOSUNG) | 19개 | ㄱ ㄲ ㄴ ㄷ ㄸ ㄹ ㅁ ㅂ ㅃ ㅅ … |
| 중성 (JUNGSUNG) | 21개 | ㅏ ㅐ ㅑ ㅒ ㅓ ㅔ ㅕ ㅖ ㅗ ㅘ … |
| 종성 (JONGSUNG) | 28개 (없음 포함) | (없음) ㄱ ㄲ ㄳ ㄴ ㄵ ㄶ ㄷ … |

## ⚙️ 오토마타 상태 흐름

```
[입력 시작]
      │
      ▼
┌─────────────┐
│ cho = None  │ ← 자음 입력 → cho 설정
└─────────────┘
      │ 자음 입력
      ▼
┌──────────────────┐
│ cho ✅, jung = None│ ← 모음 입력 → jung 설정
└──────────────────┘
      │ 모음 입력
      ▼
┌─────────────────────┐
│ cho ✅, jung ✅       │ ← 자음 입력 → jong 후보 판별
│ jong = None         │    이중모음 입력 → jung 합성
└─────────────────────┘
      │ 자음 입력
      ▼
┌──────────────────────────────────────┐
│ 다음 문자가 모음이면 → 받침이 다음 초성으로  │
│ 다음 문자가 자음이면 → 겹받침 시도 or jong  │
└──────────────────────────────────────┘
      │ 모음 입력
      ▼
┌─────────────────────┐
│ 음절 완성, 새 글자 시작│
└─────────────────────┘
```

## 🔧 함수 구조

| 함수 | 역할 |
|------|------|
| `combine_hangul(cho, jung, jong='')` | 초·중·종성 인덱스를 유니코드 공식에 대입해 완성형 한글 문자 반환 |
| `determine(ch)` | 입력 문자를 `'jaeum'`(자음) / `'moeum'`(모음) / `'etc'`(기타) 로 분류 |
| `make_hangul(line)` | 핵심 오토마타 — 문자 시퀀스를 상태 기반으로 처리해 완성된 한글 출력 |
| `main()` | CLI 루프 — Enter마다 `make_hangul()` 호출, `Ctrl+C`로 종료 |

## 🔡 특수 처리

### 이중 모음 합성 (COMPOSED_JUNG)

| 입력 순서 | 결과 | | 입력 순서 | 결과 |
|----------|------|-|----------|------|
| ㅗ + ㅏ | ㅘ | | ㅜ + ㅓ | ㅝ |
| ㅗ + ㅐ | ㅙ | | ㅜ + ㅔ | ㅞ |
| ㅗ + ㅣ | ㅚ | | ㅜ + ㅣ | ㅟ |
| ㅡ + ㅣ | ㅢ | | ㅏ + ㅣ | ㅐ |
| ㅓ + ㅣ | ㅔ | | | |

### 겹받침 합성 (COMPOSED_JONG)

| 입력 순서 | 결과 | | 입력 순서 | 결과 |
|----------|------|-|----------|------|
| ㄱ + ㅅ | ㄳ | | ㄹ + ㅂ | ㄼ |
| ㄴ + ㅈ | ㄵ | | ㄹ + ㅅ | ㄽ |
| ㄴ + ㅎ | ㄶ | | ㄹ + ㅌ | ㄾ |
| ㄹ + ㄱ | ㄺ | | ㄹ + ㅍ | ㄿ |
| ㄹ + ㅁ | ㄻ | | ㄹ + ㅎ | ㅀ |
| ㅂ + ㅅ | ㅄ | | | |

> **겹받침 + 모음** → 겹받침의 두 번째 자음이 다음 글자의 초성으로 이동  
> 예: `닭ㅇ` → `닭` + `ㅇ` 입력 시 → `달` + `고` 형태로 분리

### 백스페이스 (`<`)

```
종성 있음  → 종성 제거
중성 있음  → 중성 제거
초성 있음  → 초성 제거
위 모두 없음 → result 마지막 글자 제거
```
